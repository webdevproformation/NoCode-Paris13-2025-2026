# HTML 

- quand on se forme en informatique => le premier langage que l'on apprend c'est le html
- simple à prendre en main 
- editeur de texte (Visual Studio) + Navigateur web (firefox chrome )
- LE LANGAGE qui permet de faire fonctionner internet : Airtable / Facebook / Glide ...

# Histoire

- créé en 1991 
- en 2025 => version 5 du langage HTML5   

# Vocabulaire 

- le html est langage de balisage 

```html
<h1>bonjour les amis</h1>
```

- <h1>bonjour les amis</h1> balise html 
- qui est composée de :
    - `<h1>` => balise ouvrante
    - bonjour les amis => texte en français
    - `</h1>` balise fermante 

La balise ouvrante est composée de :
    - `<` chevron ouvrant
    - `h1` tag     => mot clé du langage HTML => headings de nvieau 1 => titre principal
    - `>` chevron fermant
