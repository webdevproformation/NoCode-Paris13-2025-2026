# webflow

- outil créé dans les années 2010
- outil qui permet de créer des sites internet de manière visuelle (faire du html / css sans écrire du html et du css)
- le concurrent de cette solution => wordpress 
- débutant et professionnel (développeur)
- modèle économique => FreeMium (Free Premium)
    - mettre à disposition plein de fonctions gratuitement
    - fonctions avancées il faut s'abonner / payer une mensualité

- se connecter très bien avec Airtable (webhook ...)
- rappel Figma / cardd
