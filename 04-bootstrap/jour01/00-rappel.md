# Base de HTML

- balise `<h1>texte</h1>`
- balise ouvrante
- balise fermante 

- dans la balise ouvrante vous avez 
- < chevron ouvrant
- h1 tag 
-  attribut class="text-danger"
- > chevron fermant

# Bootstrap 

- librairie qui permet de mettre en forme les balises html
- en fonction des besoins de mis en page je vais ajouter une ou plusieurs class dans l'attribut de la balises 

<h1 class="text-center text-primary border border-1"></h1>

le site officiel <https://getbootstrap.com>
