# Vocabulaire 

- le html est langage de balisage 

```html
<h1>bonjour les amis</h1>
```

- <h1>bonjour les amis</h1> balise html 
- qui est composée de :
    - `<h1>` => balise ouvrante
    - bonjour les amis => texte en français
    - `</h1>` balise fermante 

La balise ouvrante est composée de :
    - `<` chevron ouvrant
    - `h1` tag     => mot clé du langage HTML => headings de nvieau 1 => titre principal
    - `>` chevron fermant 
    -

# Ma shortlist des balises html à connaitre 

```html
# titre
<h1> <h2> <h3> <h4> balise qui permet de créer des titres (heading)

# le texte
<p> écrire un paragraphe

# liste 
<ul> liste à puce (Unordered List)
<ol> liste numérotée (Ordered List)
<li> élément de liste (List Item)

# mise avant 
<strong> mot important 
<em>     mot avec emphase (répéter plusieurs fois)

# image 
<img >

# google map / vidéo Youtube 
<iframe>

# formulaire
<form>
<input> => champ de saisi
<select> => menu déroulant
<textarea> => zone de texte multiligne
<button>   => bouton 

# balise de zone
<div> <span>
<header>
<main> 
<footer>
<article>
<section> 
<nav>

# tableau 
<table>
<thead> <tbody>
<tr> ligne table row 
<th> cellule de titre
<td> celule de base 

# lien hypertexte

<a> Anchor 
```
