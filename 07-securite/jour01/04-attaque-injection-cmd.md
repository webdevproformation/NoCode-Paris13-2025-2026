# injection de commande

formulaire dans lequel vous avez la possibilité de saisir une adresse IP
qui va permettre de vérifier que votre site fonctionne bien 

# liste des fichiers 

webdevpro.net | ls -al

# créer un fichier
webdevpro.net | echo 'coucou' > txt.md


# changer le nom d'un fichier
webdevpro.net | mv txt.md tutu.md

# lire le contenu d'un fichier
webdevpro.net | cat index.php
