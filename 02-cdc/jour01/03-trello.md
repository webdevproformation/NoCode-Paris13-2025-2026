# cas pratique 

- se mettre par groupe de 2 ou 3 
- créer un nouveau tableau trello (sur l'un des ordinateurs)
- vous devez créer une application de réservation de vehicule de fonction
- afin de vous organiser correctement vous décidez d'utiliser Trello pour listes l'ensemble des fonctionnalités à créer sur l'application :
- voici une première liste

- créer la base de données
- créer une page d'accueil contenant un formulaire de recherche de véhicule
- créer la page de résultat de la recherche
- verifier qu'un véhicule déjà réservé n'est pas visible dans la page de résultat
- créer la page de réservation qui contient un formulaire avec
    - nom / prénom / email 
    - un formulaire de paiement
- créer une page permettant d'ajouter un nouveau véhicule dans la flotte de véhicule
