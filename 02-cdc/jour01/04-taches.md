# Dans Trello 

- permet de créer des tableaux 
    - plusieurs colonnes (Product Back log / To Do / Finish)
    - dans les colonnes on va mettre des étiquettes 
        - décrire la tâche à réaliser 
        - 

# étiquette technique qui s'appelle User Stories

En tant que [X] 
(client / utilisateur / gestionnaire du site)

Je veux que [Y] 
(fonctionnalité à réaliser) 
Consulter / créer / modifier un vehicule de fonction

Afin de [Z] l'objectif de la fonctionnalité 
=> compléter le parc de véhicule

dans Trello / dans Back log

créer une nouvelle tâche => consulter réservation 
Dans la zone de description tu peux écrire :

---

Le client final qui a réservé un véhicule

Doit trouver le vehicule réservé dans la sa page 
profil sur le site
