# SEO

Search Engine Optimization = Référencement Naturel 

lorsque vous mettez le projet sur internet 

vos utilisateurs dispose de 3 manières d'accéder votre site 

- lemonde.fr / wikipedia / google / facebook (utilisateurs connaissent votre nom de site)
- back link / lire une page sur un site internet ET les utilisateurs cliquer sur un lien hypertexte qui va amener vers votre site 
- faire une recherche dans un moteur de recherche (google => mot clé) xxxx

LE SEO ( html ) comme remplir une page html pour influencer les résultats des recherches de google  


lorsque l'on est sur un moteur de recherche saisir des mots clés 

"sql select case when else"

plusieurs zones 

- zone payante (les webmaster vont payer google pour leur dire sql select case when else => met moi dans la zone payant)
- zone naturelle (pas payant) => google va via des algorithmes te positionner dans la zone gratuite

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <!--  il va falloir OPTIMISER le texte dans la balises du head de la page 
        title => ESSENTIELLE
        meta description
        favicon
        ... 
    -->
</head>
<body>
    <h1> 
    <h2>
    <strong> <em>
    <img alt="...">
 // ... 
</body>
</html>
``` 

Ajouter des services externes au site : 
    
- Google Analytics /  => volume d'utilisateur qui viennent sur votre site internet
- Google Search Console => voici la liste des mots sur lesquels ton site est présents dans les page de résultats de google 

Traffic de votre site internet => combien de gens viennent sur votre site 

---

fichiers robots.txt / sitemap.xml 

fichiers qui donnent des indications aux scripts (robots) de crawling de google 

<https://webdevpro.net/robots.txt> => donner des directives aux crawler sur ce qu'ils doivent stocker dans la base de données

sitemap.xml  => liste l'ensemble des pages de votre site internet et donner la date de dernière modification de chaque page
