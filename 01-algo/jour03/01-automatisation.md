# Automatisation / No Code 

- à partir des année 2010 
- sites internets qui propose des API 
- API Application Program Interface 
- pouvoir faire travailler 2 applications web sans avoir à les modifier au niveau de leur code 

# Zapier 

- site internet de type connecteur deux ou plusieurs applications web
- utiliser le meilleur de chaque application 
- améliorer la productivité

# les principaux outils en 2025

- zapier
- make.com
- n8n 

# principe de base de tous ces outils 

Application A (Gmail / Trello / Notion ....)

- Trigger / déclencheur (je viens de recevoir un email / je viens d'ajouter un nouvel onglet / tuile trello décalée dans une autre colonne )

- Action 
    - mapping (prendre les informations de email pour les injecter dans la base de données )


Application B ( Base de données AirTable / envoyer des emails serveur email  )
