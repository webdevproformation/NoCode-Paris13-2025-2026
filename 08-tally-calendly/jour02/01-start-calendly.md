# Calendly Présentation

- outil Nocode 
- permettre de faciliter la prise de rdv 
- en général pour prendre un rdv 
    - client       => son planning 
    - prestataire  => son planning 
    - échange de plusieurs emails pour déterminer un créneau ou les deux sont disponibles 
    - L'objectif de fond de Calendly est d'éliminer les échanges d'email pour prendre un rdv à plusieurs 
- outil européen => Hollandais
- se synchronise avec outil nocode => Google agenda 
- ET si vous utiliser Zapier et Airtable 
- calendly => CRM ( Customer Relationship Management ) => Hubspot / SaleForce 
- Dans la version payante : vous pouvez faire payer le rdv (stripe)

# Principe général 

- vous (utilisateur) => définir vos disponibilités 
- facultatif => synchroniser avec votre agenda 
- créer événenement (tutorat)
- créer un lien (que l'on soit envoyer par email / iframe)
- l'invité choisir un créneau
- calendly va 
    - bloquer le créneau (dans votre agenda)
    - envoyer une confirmation à l'invité
    - relancer / automatisation
