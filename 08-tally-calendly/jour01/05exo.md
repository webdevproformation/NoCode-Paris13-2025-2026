Vous travaillez dans une société de recrutement
vous êtes à la recherche de nouveaux talents pour le compte de vos clients
Et vous souhaitez créer un formulaire de demande de contacts qualifiés


ce formulaire est composé de 3 pages 

page 1 Profil du demandeur


Champs à créer :
- Prénom (obligatoire)
- Nom (obligatoire)
- Email (obligatoire)
- Type de profil (choix unique) :
  - Particulier
  - Indépendant / Freelance
  - Entreprise

---

page 2 Besoin & projet

Champs à créer :
- Objet de la demande (liste déroulante) :
  - Demande d’information
  - Devis
  - Support
- Description du besoin (champ texte long)

---

page 3 Confirmation & envoi

Contenu à afficher :
- Message de confirmation :
  > *Merci pour votre demande. Nous reviendrons vers vous sous 48h.*

- Case à cocher obligatoire :
  - “J’accepte d’être recontacté dans le cadre de cette demande”
