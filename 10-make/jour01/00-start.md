# Make.com 

support : <https://formation.webdevpro.net/nocode-make/>

# what is it ??

- concurrent de Zapier
- outil no code qui permet de lier des outils no code (Notion / Airtable / Gmail / Google Sheet ...)
- par défaut il est livré avec 2 000 applications 
- Zapier  => mapping (prendre une donnée dans l'application trigger => l'utiliser dans l'application Action)
- Make non seulement on peut réaliser des mapping MAIS en + : appliquer des transformations avancées 
- vous avez de nombreux modules interne de l'outil qui permettent de réaliser des opérations / traitements avec du code
    - CSV / Excel 
    - HTTP => `requests`
    - créer des variables 
    - ... 
    - IA 

# Vocabulaire

Zapier 
    trigger
    action 
=> Zap (un automatisation entre deux outils)

Make.com 
    Automatisation => Scénario 
    Modules qui vont s'enchaîner l'un après l'autre
    Visuellement ça ressemble à des cercles qui vont se mettre l'un après l'autre 
    Le 1er module du scénation => Trigger 
    Outil très visuel facile à prendre en main 

# Paiement 

Zapier => Zap qui contient trigger + action => coûté 1  
          Remise à zéro mensuelle (100 exécutions / mois)
          Lors de la phase de création des automatisations c'est payant

Make.com => Scénario contient plusieurs modules 
         => si dans un scénario vous avez 10 modules => coûte 10 
         => plan gratuit 1000 unités mensuel 
         => lors de la phase de création tous les tests sont décomptés
