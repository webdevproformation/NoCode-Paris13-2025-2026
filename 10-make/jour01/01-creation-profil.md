# créer un profil sur l'outil 

- <https://www.make.com/en>
- Cliquer sur le bouton `Get Started Free`
- créer un profil
- pour finaliser la création de profil => email de vérification à valider (il peut mettre un peu de temps à arriver)


# Vocabulaire

- Scénario  => automatisation complète
- Module    => étape du scénario
- Trigger   => 1er module du scénario
- Action    => autres modules du scénario
- Route     => branch de scénario
- Bundle    => Informations qui sont transmises / partagées en les modules
- Opération => unité de facturation (décompte de forfait d'utilisation)
